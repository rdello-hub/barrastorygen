# Stories Generator

A blazing-fast, premium web-based Instagram Stories builder designed for financial consultants. It allows rapid creation of 1080x1920 high-conversion visual assets.

## Core Features (Phase 1 MVP)
- **WYSIWYG Editing**: Edit 1080x1920 visuals directly on the canvas.
- **Layout Presets**: Standardized, conversion-focused templates.
- **Browser-only**: Zero backend. Uses LocalStorage and DOM-to-PNG export.
- **Brand Aligned**: Strictly uses the defined Dark Navy/Royal Blue/Pale Yellow color scheme.

## Future Roadmap (Phase 2)
An integrated AI co-pilot for automated copy generation (P.A.S.A/A.I.D.A frameworks) and intelligent layout variations.
